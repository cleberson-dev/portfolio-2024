My new portfolio for 2024. A little bit shorter, more concise.

## Run the project

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

- Typescript (v5)
- Tailwind CSS (v3)
- ESLint (v8)
- Vitest (v1.0.4)
- React Testing Library (v14.1.2)
